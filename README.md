# Youtube Clone

Youtube clone created using React Js, Material UI5 and Rapid API

# Functions

- Browse Channel
- Play Videos
- Search Videos

# Download

```
npm install --legacy-peer-dep

```

# Tech Stack

**Tools Used** - React.js, Materual UI5, Rapid API
